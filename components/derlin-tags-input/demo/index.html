Basic usage

Use the tags attribute to bind tags to a custom property. You can choose between one-way or two-ways binding.

Delimiters

Use the delimiter attribute to control which key triggers the tag processing (default to a comma).

SIze and label

By default, derlin-tags-input takes all the available space (width: 100%). Use the expand attribute to start small and let the input expand as needed. You can also disable the floating label with the no-floated-label attribute:
